    • Sonify | Full Stack

    ×


    Sonify was developed as a Capstone Senior Design Project by a team of three engineering students at the University of Miami. The Full Stack Web Application provides recommendations based off of the acoustic properties of a given track. Winners: Best Overall Senior Design Project & Industry Impact Award

    More Info

    By interfacing with Spotify's API, User's can search for recommendations of any track in Spotify's catalog. The track's audio file is then analyzed on the Back End where 12 features such as tempo and chord progressions are extracted. We applied machine learning to provide recommendations, feeding our feature vectror into a Weighted KNN algorithm and using the Wave-Hedges distance metric to recommend the 10 nearest neighbors.

    Then, by making AJAX calls to Spotify's API and fetching album art and audio previews, the results are then presented on the Front End to the user. For this project, my teammates primarily focused on implementing our Backend Analysis & Recommendation algorithms while I built out our Full Stack with Django and Front End in Javascript, AJAX, HTML, and CSS. Because the backend algorithms take some time to process User input, I needed to make sure that the User Experience was a cohesive pleasant experience even when waiting for results. To this end, I designed a logo with multiple Vector Layers in Photoshop which I then animated using CSS to create a Loading animation which was conditionally rendered on the Front End.

    • Music Visualizer

    ×
    Demo the app


    This music visualizer was made using JavaScript, Three.js, and the Web Audio API. Watch as the earth and its surrounding colorful sound waves react to your uploaded track. By extracting and modulating the frequencies of the user's input song, vertices of the rendered objects are displaced in realtime in 3D space.

    Future improvements include optimizations, functionality to customize the color and number of outer rings, and incorporating the Spotify API to allow user's to view visualization of any tracks in the Spotify database.

    My last year at Miami, I developed a 2D top-down Ski game in Unity. I designed all game assets & sprites in Photoshop and coded the game logic in C#.
