    • Sonify | Full Stack

    ×

    Sonify was developed as a Capstone Senior Design Project by a team of three engineering students at the University of Miami. The Full Stack Web Application provides recommendations based off of the acoustic properties of a given track.

    More Info

    By interfacing with Spotify's API, User's can search for recommendations of any track in Spotify's catalog. The track's audio file is then analyzed on the Back End where 12 features such as tempo and chord progressions are extracted. We applied machine learning to provide recommendations, feeding our feature vectror into a Weighted KNN algorithm and using a Wave-Hedges distance metric to recommend the 10 nearest neighbors.

    By making AJAX calls to Spotify's API and fetching album art and audio previews, the results are then presented on the Front End to the user. For this project, my teammates primarily focused on implementing our Backend Analysis & Recommendation algorithms while I built out our Full Stack with Django and Front End in Javascript, AJAX, HTML, and CSS. Because the backend algorithms take some time to process User input, I needed to make sure that the User Experience was a cohesive pleasant experience even when waiting for results. To this end, I designed a logo with multiple Vector Layers in Photoshop which I then animated using CSS to create a Loading animation which was conditionally rendered on the Front End.
